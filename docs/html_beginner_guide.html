HTML 기초 가이드

초등학생도 쉽게 배우는 웹페이지 만들기!

🤔 HTML이 뭔가요?

HTML은 웹사이트의 뼈대를 만드는 언어예요!

마치 레고를 쌓을 때 "이 블록은 여기에, 이 블록은 저기에" 하고 설명하듯,

HTML은 "이 글자는 제목이야", "이 그림은 여기 넣어" 라고 컴퓨터에게 알려주는 거예요.

🥪 HTML 문서 = 샌드위치랑 비슷해요!

HTML 문서는 빵-속-빵 구조예요. 샌드위치 생각하면 쉬워요!

🍞 <html> 태그 (문서 전체를 감싸요)
🥬 <head> 태그 (보이지 않는 정보)
🥩 <body> 태그 (실제로 보이는 내용)
🍞 </html> 태그 (문서 끝!)

📋 기본 구조를 한눈에 보기

<!-- 이 문서는 HTML5 문서예요! -->
<!DOCTYPE html>
<html>
<head>
    <title>나의 첫 웹페이지</title>
</head>
<body>
    안녕! 나는 처음 만든 웹페이지야!
</body>
</html>
부분 설명
<!DOCTYPE html> "이 문서는 HTML5야!" 라고 컴퓨터에게 알려줘요
<html> 문서의 시작! 모든 것을 감싸는 큰 상자예요
<head> 머리 부분. 화면에 안 보이는 설정들이 들어가요
<title> 브라우저 탭 제목이에요. 예: "나의 첫 웹페이지"
<body> 몸통 부분. 여기에 넣은 것만 화면에 보여요!
</body> 몸통 끝
</html> 문서 끝
  • 태그는 <태그>로 열고 </태그>로 닫아요!
  • 예: <body> (열기) → </body> (닫기)
  • 열고 닫는 태그 사이에 내용을 넣어요!

🎮 직접 따라 해보기!

📝 실습 방법

메모장(또는 VS Code)에 아래를 그대로 써보세요:

1️⃣ 메모장을 열어요
2️⃣ 아래 코드를 그대로 복사해서 붙여넣어요
3️⃣ 파일 → 다른 이름으로 저장 → 이름을 index.html 로 저장!
4️⃣ 저장한 파일을 더블클릭하면 브라우저에서 웹페이지가 쫙 펼쳐져요! 🎉
<!DOCTYPE html>
<html>
<head>
    <title>내 홈페이지</title>
</head>
<body>
    <h1>안녕하세요!</h1>
    <p>저는 HTML을 배우고 있어요.</p>
    <p>오늘은 2026년 5월 14일이에요.</p>
</body>
</html>

🏷️ 자주 쓰는 태그 쉽게 외우기

태그 뜻 예시
<h1> Heading 1 (큰 제목) <h1>책 제목</h1>
<h2> Heading 2 (작은 제목) <h2>1장</h2>
<p> Paragraph (단락, 문단) <p>이야기 내용</p>
<br> Break (줄바꿈) 문장 끝에 쓰면 다음 줄로!
<img> Image (그림) <img src="고양이.jpg">
<a> Anchor (링크, 닻) <a href="https://naver.com">네이버 가기</a>

✅ HTML 작성 체크리스트

📋 꼭 확인해야 할 것들!

☑️ <!DOCTYPE html> 선언이 들어갔나요?
☑️ <html> 태그로 시작하고 </html>로 끝났나요?
☑️ <head>와 <body>가 모두 있나요?
☑️ <title> 태그로 페이지 제목을 지정했나요?
☑️ 모든 태그를 열고 닫았나요? (예: <body> → </body>)

🎯 한 문장으로 정리!

HTML은 웹페이지의 "설계도"예요.

<html>로 시작해서 </html>로 끝내고,
안에는 <head>(설정)와 <body>(보이는 내용)를 넣으면 돼요!
