# Gemini CLI 실행 도구 가이드

원하는 경로에서 Gemini CLI를 쉽게 실행할 수 있도록 두 가지 방법을 제공합니다.

## 방법 1: 우클릭 메뉴에 추가 (추천)
윈도우 탐색기에서 폴더 빈 공간을 우클릭했을 때 "Gemini CLI 여기서 열기" 메뉴가 나타나게 합니다.

1. `add-to-context-menu.ps1` 파일을 마우스 우클릭합니다.
2. "PowerShell에서 실행"을 클릭합니다.
3. 이제 어느 폴더에서든 빈 공간을 우클릭하여 Gemini CLI를 바로 실행할 수 있습니다.

## 방법 2: 실행 파일 사용
특정 폴더에 넣어두고 더블 클릭하여 실행하는 방식입니다.

1. `gemini-here.bat` 파일을 원하는 폴더로 복사합니다.
2. 해당 파일을 더블 클릭하면 해당 경로에서 Gemini CLI가 시작됩니다.

---
**참고:** `gemini` 명령어가 시스템 PATH에 등록되어 있어야 정상적으로 작동합니다. (이미 설치되어 있으므로 보통 바로 작동합니다.)
