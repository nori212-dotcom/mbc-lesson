🌈 CSS class vs id

누구나 쉽게 배우는 별명과 이름표 이야기!

🆔 ID (아이디) : 세상에 딱 하나뿐인 이름표!

👤

ID는 여러분의 이름이나 주민등록번호와 같아요!
우리 학교에 여러분과 똑같은 이름과 번호를 가진 친구가 또 있으면 안 되겠죠?

  • ✅ 한 페이지에 딱 한 번만 쓸 수 있어요.
  • ✅ CSS에서는 이름 앞에 # (샵)을 붙여서 불러요.
  • ✅ "이건 진짜 특별해!"라고 강조하고 싶을 때 사용해요.
#my-name {
  color: gold;
}

<p id="my-name">저는 세상에 하나뿐인 주인공이에요!</p>

🏷️ Class (클래스) : 우리 팀 다 같이 입는 유니폼!

⚽

Class는 축구팀 유니폼이나 반 친구들의 별명과 같아요!
여러 명의 친구가 똑같은 유니폼을 입고 "우리는 1반 팀이야!"라고 할 수 있죠?

  • ✅ 여러 번 반복해서 마음껏 쓸 수 있어요.
  • ✅ CSS에서는 이름 앞에 . (점)을 붙여서 불러요.
  • ✅ "우리 다 같이 이 색깔로 바꿀까?" 할 때 사용해요.
.team-blue {
  color: blue;
}

<p class="team-blue">우리는 파랑 팀 1번!</p>
<p class="team-blue">우리는 파랑 팀 2번!</p>

🆚 한눈에 비교하기

특징 🆔 ID (아이디) 🏷️ Class (클래스)
몇 번 써요? 딱 한 번! 여러 번 반복 OK!
기호 # (샵) . (점)
비유 주민등록번호, 내 이름 학교 교복, 팀 유니폼
힘(우선순위) 매우 강력함! 💪 보통이에요 😊
💡 개발자 선생님의 꿀팁!

보통 웹사이트를 만들 때는 Class를 더 많이 사용해요. 왜냐하면 나중에 똑같은 스타일을 다른 곳에서도 또 쓰고 싶어질 때가 많거든요! ID는 정말로 딱 하나만 있어야 하는 중요한 부분(예: 메뉴 바 전체 상자)에만 아껴서 사용한답니다.

🌟 ID와 Class만 잘 써도 여러분은 이미 꼬마 개발자! 🌟
