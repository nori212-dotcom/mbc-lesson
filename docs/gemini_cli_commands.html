Gemini CLI 명령어 가이드

Gemini CLI에서 사용할 수 있는 슬래시(/) 명령어 전체 목록

1. 시스템 및 일반 도구

help (또는 /?)
사용 가능한 모든 명령어와 상세 사용법을 출력합니다.
about
현재 CLI 버전 및 시스템 정보를 표시합니다. 이슈 보고 시 유용합니다.
docs
웹 브라우저를 통해 공식 도움말 문서를 엽니다.
stats
현재 세션의 상세 통계(사용 시간, 토큰 사용량, 도구 호출 횟수 등)를 확인합니다.
clear
터미널 화면을 깨끗이 지웁니다. Ctrl + L
quit (또는 /exit)
CLI를 종료합니다. --delete 옵션을 붙이면 세션 기록을 즉시 삭제합니다.

2. 대화 세션 및 기록 관리

resume (또는 /chat)
과거 세션을 탐색하거나 다시 시작합니다.
• list: 체크포인트 목록 보기
• save <tag>: 대화 상태 저장
• share: Markdown/JSON으로 내보내기
rewind
대화 기록을 뒤로 탐색하여 특정 시점으로 채팅 상태와 파일 변경 사항을 모두 되돌립니다. Esc 2번
restore [id]
도구가 실행되기 전의 파일 상태로 프로젝트를 복원합니다.
compress
이전 대화 문맥을 요약본으로 압축하여 토큰 사용량을 최적화합니다.

3. 워크스페이스 및 환경 설정

settings
CLI의 동작과 외관을 변경할 수 있는 설정 편집기를 엽니다.
directory (또는 /dir)
워크스페이스에 포함할 디렉토리를 추가하거나 확인합니다.
memory
GEMINI.md 파일에 정의된 프로젝트 문맥을 관리합니다.
• refresh: 설정 다시 불러오기
• show: 현재 참고 문맥 표시
theme
CLI 색상 테마를 변경하는 다이얼로그를 엽니다.
vim
입력창에서 Vim 스타일의 네비게이션과 편집 모드를 설정합니다.

4. 모델 및 도구 제어

model
사용할 Gemini 모델(Pro, Flash 등)을 선택하거나 설정합니다.
tools
현재 활성화된 도구 목록을 보여줍니다. desc 옵션으로 상세 설명을 볼 수 있습니다.
mcp
Model Context Protocol 서버를 관리합니다. (목록 확인, 재시작, 인증 등)
plan
'계획 모드'로 전환하여 수행할 작업 단계를 미리 검토합니다.

5. 확장 기능 및 스킬 관리

extensions
플러그인을 설치, 업데이트하거나 관리합니다.
skills
특정 작업에 최적화된 에이전트 스킬을 활성화하거나 목록을 확인합니다.
agents
로컬 또는 원격에서 작동하는 서브에이전트를 관리합니다.
commands
사용자 정의 커스텀 명령어(.toml)를 관리하고 새로 고침합니다.

6. 개발자 및 특수 도구

copy
마지막 AI 응답 결과를 클립보드에 즉시 복사합니다.
init
현재 프로젝트를 위한 GEMINI.md 가이드 파일 생성을 도와줍니다.
setup-github
GitHub 이슈 트리아지 및 PR 리뷰 자동화 설정을 진행합니다.
shells (또는 /bashes)
백그라운드에서 실행 중인 셸 프로세스들을 모니터링합니다.
