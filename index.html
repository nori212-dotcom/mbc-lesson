프로젝트 요약

이 프로젝트는 HTML/CSS 학습용 웹사이트로, 세 개의 주요 영역을 포함합니다.

  • 문서 가이드: docs/ 폴더에 HTML 기본 구조, 웹 명명 규칙, 블록/인라인 설명, Gemini CLI 명령어 가이드가 있습니다.
  • HTML 실습: 01/ 폴더에는 HTML 기본 실습과 퀴즈, 과제 파일이 있습니다.
  • CSS 실습: 02/ 폴더에는 CSS 실습 과제와 퀴즈 파일, 스타일 예제가 있습니다.
  • 추가 예제: 03/ 폴더는 커뮤니티, 상세 페이지, 자바스크립트 예제와 프롬프트 파일을 포함합니다.
  • 기타: ex/ 폴더는 QR, 설정 완료, 배너 디자인 등 부가 콘텐츠를 담고 있습니다.

이 페이지는 학습 자료와 실습 결과를 한곳에서 쉽게 찾을 수 있도록 구성되어 있습니다.
